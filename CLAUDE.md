# Agent Factory 项目规范

## 代码规范

- 所有代码必须有中文注释
- 使用 Python type hints
- 异常处理必须完善
- 测试覆盖率 > 80%
- 使用 Pydantic 进行数据验证

## 安全规范

- 所有 LLM 输出必须经过 Guardrails 验证
- 工具调用必须有权限控制
- 敏感数据必须脱敏
- API 密钥使用 .env 管理，不要硬编码

## 文档规范

- 所有文档使用中文
- 代码示例必须可运行
- 概念解释使用生活比喻
- 每个概念都要有 Obsidian 链接

## Git 规范

- commit message 使用中文
- 每个功能一个分支
- PR 必须有描述
- 代码合并前必须通过测试

## 学习规范

- 每天最多 3 件事
- 代码必须能运行才能进入下一天
- 卡住了主动反馈，不要硬撑
- 复盘比赶进度更重要

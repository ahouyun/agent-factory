# Agent Factory 🤖

> 从零到精通的 AI Agent 系统学习课程

## 项目简介

Agent Factory 是一套 20 周的 AI Agent 学习课程，融合了 10 个优秀开源项目的内容，帮助零基础学习者系统掌握 AI Agent 开发。

## 课程特色

- **20 周系统学习**: 从 Python 基础到完整 Agent 项目
- **每日任务驱动**: 11 段式结构化任务，每天聚焦 2-3 件事
- **反馈自适应**: P0-P4 五级优先级，根据学习状态自动调整
- **10 个来源融合**: 去重后整合最佳内容
- **面试导向**: 每周内置面试模拟题，学完即可面试

## 学习路线

```
Phase 0 (Week 0)     → 预备周：环境搭建
Phase 1 (Week 1-4)   → 地基：Python + FastAPI + LLM 原理
Phase 2 (Week 5-9)   → 核心：Agent 范式 + RAG + 框架 + 安全 + 记忆
Phase 3 (Week 10-13) → 进阶：RL + 评估 + 部署 + 选修
Phase 4 (Week 14-17) → 综合项目（3 选 1）
Phase 5 (Week 18-20) → 毕业冲刺：部署 + 简历 + 面试
```

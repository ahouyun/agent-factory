## 🎯 项目介绍

&emsp;&emsp;2025 年是"Agent 元年"，技术焦点正从训练更大的基础模型，转向构建更聪明的智能体应用。然而，当前系统性、重实践的 Agent 学习教程却极度匮乏。为此，我们发起了 <strong>Agent Factory</strong> 项目，融合了 <strong>ai-agent-daily-mentor</strong>、<strong>hello-agents</strong>、<strong>happy-llm</strong>、<strong>Microsoft AI Agents</strong>、<strong>OpenAI Agents SDK</strong> 等 10 个优秀开源项目的精华，经过去重和优化后，形成了一套完整的 20 周学习路径。

&emsp;&emsp;Agent Factory 旨在带领你从 Python 零基础出发，系统掌握 AI Agent 开发的全流程。教程将从工程化基础讲起，逐步深入 Agent 范式、RAG 检索增强、安全防护、记忆系统、通信协议等核心技术，最终通过一个完整的 Agent 项目实战，让你具备求职面试的能力。我们相信，最好的学习方式就是动手实践。希望这本教程能成为你探索智能体世界的起点，能够从一名大语言模型的"使用者"，蜕变为一名智能体系统的"构建者"。

## 💡 如何学习

&emsp;&emsp;欢迎你，未来的智能系统构建者！在开启这段旅程之前，请允许我们给你一些清晰的指引。

&emsp;&emsp;本项目兼顾理论与实战，旨在帮助你系统性地掌握从单个智能体到多智能体系统的设计与开发全流程。尤其适合<strong>零基础转行者</strong>、<strong>有 Python 基础的开发者</strong>、<strong>在校学生</strong>以及对 AI Agent 技术抱有浓厚兴趣的<strong>自学者</strong>。在学习本项目之前，我们希望你具备基础的计算机操作能力，Python 基础会在 Week 0-1 帮你补齐。

&emsp;&emsp;项目分为五大部分，每一部分都是通往下一阶段的坚实阶梯：

- <strong>Phase 0: 预备周</strong>（Week 0），我们将帮你搭建完整的开发环境，包括 Python、Git、VS Code、Claude Code 等工具，并快速掌握 Python 基础。

- <strong>Phase 1: 地基</strong>（Week 1-4），你将掌握 Python 工程化、FastAPI 后端开发、LLM 原理与 API 调用，以及 Agent 的经典范式（ReAct、Plan-and-Solve、Reflection、Metacognition）。

- <strong>Phase 2: 核心能力</strong>（Week 5-9），这是动手实践的核心阶段。你将深入学习 RAG 检索增强、Agent 框架（LangChain/LangGraph/CrewAI）、安全防护（Guardrails）、记忆系统（MemGPT）、通信协议（MCP/A2A/NLWeb），并完成一个项目 MVP。

- <strong>Phase 3: 进阶专精</strong>（Week 10-13），你的 Agent 将"学会"训练与优化。我们将深入探索 Agentic RL、评估体系、生产部署，并提供 9 个选修方向供你按兴趣选择。

- <strong>Phase 4-5: 项目实战与毕业</strong>（Week 14-20），你将把所学融会贯通，完成一个完整的 Agent 项目，并进行项目打磨、简历准备和模拟面试。

&emsp;&emsp;智能体是一个飞速发展且极度依赖实践的领域。为了获得最佳学习效果，我们<strong>强烈建议你将理论与实践相结合</strong>，亲手运行、调试甚至修改项目里提供的每一份代码。

&emsp;&emsp;现在，准备好进入智能体的奇妙世界了吗？让我们即刻启程！

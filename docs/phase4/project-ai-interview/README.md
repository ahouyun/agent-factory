# 项目一：AI-Interview 智能模拟面试系统

## 项目概述

基于 RAG + Agent 的智能模拟面试系统，支持简历解析、岗位匹配、题库检索、AI 面试、评分报告。

## 技术栈

- **后端**: FastAPI + SQLAlchemy + Pydantic
- **RAG**: ChromaDB + LangChain
- **Agent**: LangGraph + Function Calling
- **实时通信**: WebSocket
- **安全**: Guardrails + JWT
- **部署**: Docker Compose

## 项目规模

- 代码量: 约 3000-5000 行
- 开发周期: 4 周（Week 14-17）

## 核心功能

1. **简历解析**: 上传 PDF/Word，提取关键信息
2. **岗位匹配**: 根据简历匹配目标岗位
3. **题库检索**: RAG 检索相关面试题
4. **AI 面试**: LangGraph 驱动的面试流程
5. **自动评分**: 多维度评分 + 反馈
6. **面试报告**: 生成详细的能力评估报告

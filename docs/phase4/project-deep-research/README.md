# 项目二：DeepResearch 深度研究 Agent

## 项目概述

多 Agent 协作的深度研究系统，自动规划研究路径、多源检索、交叉验证、生成研究报告。

## 技术栈

- **多 Agent**: LangGraph + AutoGen
- **检索**: 搜索引擎 API + 向量数据库
- **文档处理**: LangChain Document Loaders
- **任务委派**: Handoff 模式

## 项目规模

- 代码量: 约 2000-3000 行
- 开发周期: 4 周（Week 14-17）

## 核心功能

1. **研究规划**: 自动分解研究问题为子任务
2. **多源检索**: 搜索引擎 + 学术论文 + 新闻
3. **信息整合**: 交叉验证 + 去重 + 排序
4. **报告生成**: 结构化研究报告输出
5. **引用管理**: 自动添加引用来源

## Agent 架构

```
用户问题
    ↓
[规划 Agent] → 研究计划
    ↓
[检索 Agent x3] → 多源检索结果
    ↓
[验证 Agent] → 交叉验证
    ↓
[整合 Agent] → 最终报告
```

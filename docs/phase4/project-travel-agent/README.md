# 项目三：多智能体旅行助手

## 项目概述

基于 MCP + CrewAI 的多智能体旅行助手，行程规划 Agent + 订票 Agent + 推荐 Agent 协作。

## 技术栈

- **多 Agent**: CrewAI + MCP
- **外部 API**: 天气/航班/酒店 API
- **通信协议**: MCP Server

## 项目规模

- 代码量: 约 2000-3000 行
- 开发周期: 4 周（Week 14-17）

## 核心功能

1. **行程规划**: 根据偏好自动规划行程
2. **订票服务**: 航班/酒店查询和预订
3. **推荐系统**: 景点/餐厅推荐
4. **实时更新**: 天气/交通变化通知

## Agent 架构

```
用户需求
    ↓
[规划 Agent] → 行程方案
    ↓
[订票 Agent] → 航班/酒店
[推荐 Agent] → 景点/餐厅
    ↓
[协调 Agent] → 整合结果
```

⭐⭐⭐⭐ 多 Agent/协议/集成

# Week 0 Day 4: Obsidian 知识库搭建

> 今天我们将安装 Obsidian 并搭建学习笔记体系。Obsidian 是一个强大的知识管理工具，支持双向链接，能帮你把零散的知识点串成知识网络。

---

## 一、为什么需要 Obsidian？

学习过程中，你会接触大量新概念、新代码、新工具。如果没有一个好用的笔记工具，这些知识会很快遗忘。

**Obsidian 的核心优势：**

1. **双向链接**：用 `[[笔记名]]` 创建笔记之间的连接，形成知识网络
2. **本地存储**：所有笔记都是 Markdown 文件，保存在你自己的电脑上
3. **免费使用**：核心功能完全免费
4. **跨平台**：Windows、Mac、Linux 都能用
5. **插件丰富**：社区有大量实用插件

**生活比喻：** 如果普通的笔记 app 像是一个个独立的便利贴，Obsidian 就像一面贴满便利贴并用线连起来的白板——你能看到知识点之间的关系。

## 二、安装 Obsidian

### 2.1 下载安装

1. 打开浏览器，访问 https://obsidian.md/
2. 点击 "Download" 按钮
3. 选择 Windows 版本下载
4. 运行下载的安装程序
5. 安装完成后，Obsidian 会自动启动

### 2.2 创建 Vault（知识库）

Vault 是 Obsidian 中的一个"知识库"，本质上就是一个文件夹。

1. Obsidian 启动后，点击 "Create new vault"
2. Vault 名称输入：`Agent Factory 学习笔记`
3. 存储位置选择：`D:\claude-workspace\`（或者其他你喜欢的位置）
4. 点击 "Create"

创建完成后，你会看到一个空白的 Obsidian 界面。

模板可以让你快速创建格式统一的笔记。

### 4.2 使用模板创建新笔记

1. 在 "01-每日笔记" 文件夹中，右键 -> "New note"
2. 在新笔记中输入 `---` 然后回车，会自动触发模板（如果 Obsidian 没有自动填充，可以按 `Ctrl+P`，搜索 "Template: Insert template"）
3. 或者手动复制模板内容到新笔记中

## 五、双向链接（Wiki Links）

双向链接是 Obsidian 最强大的功能。它让你的笔记形成一个知识网络。

### 5.1 基本语法

```markdown
# 创建链接
[[笔记名称]]

# 带显示文本的链接
[[笔记名称|显示的文字]]

# 链接到标题
[[笔记名称#标题]]
```

### 5.2 实际示例

创建一个概念笔记 "Python 基础.md"：

```markdown
# Python 基础

## 变量

Python 中变量不需要声明类型，直接赋值即可：

```python
name = "张三"      # 字符串
age = 25           # 整数
height = 1.75      # 浮点数
is_student = True  # 布尔值
```

## 函数

函数用 `def` 关键字定义：

```python
def greet(name: str) -> str:
    return f"Hello, {name}!"
```

## 相关概念

- [[数据类型]]
- [[面向对象编程]]
- [[Git 版本控制]]
```

### 5.3 知识图谱

Obsidian 有一个"图谱"视图（Graph view），可以可视化显示所有笔记之间的链接关系。

1. 点击左侧边栏的"图谱"图标
2. 你会看到一个网络图，每个节点是一篇笔记，每条线是一个链接
3. 笔记越多，图谱越丰富

## 六、Obsidian 实用功能

### 6.1 标签

用 `#标签名` 给笔记打标签：

```markdown
---
tags: [Python, 基础, Week0]
---

# Python 基础

#Python #基础 #Week0
```

### 6.2 Callout（高亮块）

```markdown
> [!info] 信息
> 这是一个信息提示

> [!tip] 提示
> 这是一个有用的提示

> [!warning] 警告
> 这是一个警告

> [!question] 问题
> 这是一个待解决的问题
```

### 6.3 快捷键

| 快捷键 | 作用 |
|--------|------|
| `Ctrl+P` | 打开命令面板 |
| `Ctrl+N` | 新建笔记 |
| `Ctrl+E` | 编辑/预览切换 |
| `Ctrl+O` | 快速打开笔记 |
| `Ctrl+G` | 打开图谱 |

## 九、常见问题

**Q: Obsidian 打不开 Vault 怎么办？**
A: 检查 Vault 路径是否正确，确保文件夹有读写权限。如果是从外部硬盘打开，确保硬盘已连接。

**Q: `[[]]` 链接不生效？**
A: 确保在编辑模式（不是预览模式）下输入。输入 `[[` 后会自动弹出候选笔记列表。

**Q: 笔记太多找不到怎么办？**
A: 按 `Ctrl+O` 打开快速搜索，输入关键词即可找到。

**Q: Obsidian 数据会丢失吗？**
A: Obsidian 的数据就是普通的 Markdown 文件，保存在你的本地磁盘上。只要不手动删除文件夹，数据不会丢失。建议定期备份 Vault 文件夹。
